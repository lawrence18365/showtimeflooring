# Showtime Event Flooring - Style Guide

## Brand Overview
**Archetype**: The "Magical Foundation"
**Vibe**: Coastal Elegance. Sophisticated, timeless, relaxed.

## Color Palette

### Primary
- **Crisp White**: `#FFFFFF` (Backgrounds, Cards)
- **Champagne Sand**: `#F5F0E6` (Section Backgrounds, subtle warmth)

### Secondary
- **Soft Timber**: `#C1A78E` (Text accents, borders)
- **Warm Grey**: `#4A4A4A` (Body Text - softer than black)

### Accent
- **Muted Gold**: `#D4AF37` (CTAs, Key Highlights)
- **Deep Ocean Blue**: `#1B3B5A` (Footer, Trust signals)

## Typography

### Headlines
**Font**: Playfair Display (Serif)
**Weights**: 400, 700
**Usage**: Main H1, Section H2s.
**Example**: "Elevate Your Event"

### Body
**Font**: Montserrat (Sans-Serif)
**Weights**: 300, 400, 500
**Usage**: Paragraphs, Navigation, Buttons.

## UI Elements

### Buttons
- **Primary**: Solid Muted Gold (#D4AF37) with White Text.
- **Secondary**: Transparent with Deep Ocean Blue border & text.
- **Hover**: Slight opacity drop + transform translateY(-2px).

### Spacing
- **Container**: Max-width 1200px.
- **Section Padding**: 80px top/bottom (Desktop), 60px (Mobile).
